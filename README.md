# Python商品信息管理系统 python355

## 项目概述

Python商品信息管理系统是一个基于Python Flask框架的全功能后台管理系统。该系统集成了登录、注册、采购进货、采购退货、商品信息管理、供应商管理、销售管理、仓库管理、数据统计、管理员管理以及系统管理等模块，致力于提高商品管理的效率和准确性。

## 技术栈

- **后端语言与框架**：Python 3.0以上，Flask
- **前端技术**：HTML、CSS、JavaScript、jQuery
- **数据库**：MySQL 5.7
- **开发工具**：PyCharm、Navicat

## 功能模块

- **用户模块**：登录、注册
- **采购模块**：采购进货、采购退货
- **商品管理**：商品信息管理
- **供应商管理**：供应商信息管理
- **销售管理**：销售信息管理
- **仓库管理**：仓库信息管理
- **数据统计**：销售与库存数据统计
- **管理员模块**：管理员管理、系统设置

## 系统角色

- **普通用户**：进行登录、注册，使用基础功能
- **管理员**：管理用户、商品、供应商信息，处理销售与仓库数据，维护系统设置

## 运行环境

- **操作系统**：兼容Windows、Linux等操作系统
- **Python版本**：Python 3.0以上
- **数据库**：MySQL 5.7

## 部署步骤

1. 安装Python 3.0以上版本
2. 安装MySQL 5.7并创建对应数据库
3. 使用PyCharm或命令行工具，运行系统

```
商品信息管理系统/

```
